Утилита time служит для установки часов ZX Evolution и просмотра времени.
Для АТМ2+ Следует использовать  time2.
Позволяет  указать время как вручную, так и получить по NTP из сети.

Параметры командной строки
	-H	Выводит краткую справку
	-T	Установить время.	(-T17:59:38)
	-D	Установить дату		(-D21-06-2019)
	-N	Указать пользовательский NTP сервер
		по-умолчанию: (-N2.ru.pool.ntp.org)
	-Z	Установить часовой пояс
		по-умолчанию: 		(-Z3)
	-i	Получить время из сети
