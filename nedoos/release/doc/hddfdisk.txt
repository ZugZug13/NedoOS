Данная утилита позволяет разбить жёсткий диск на разделы и отформатировать их.
Для этого, после запуска, необходимо выбрать ваше устройство из списка, путем нажатия кнопок 0...4 :
		Select drive:
		[0] Nemo master
		[1] Nemo slave
		[2] ATM master(not tested!)
		[3] ATM slave(not tested!)
Далее выбрать неразмеченную облать empty :
		 N   Type      Size      Offset  PartN
		[1]: FAT           60          1   0
		[2]: FAT           70         61   1
		[3]: empty        169        131
Будет выведена подсказка:
Selected volume [3]
[C]reate
Нажатием кнопки С запустится диалог разметки раздела:
		Create partition. Enter values in megabytes
		offset [131]
		size [169]
offset - начало раздела в мегабайтах(отсчёт от начала жёсткого диска), в квадратных скобках значение по умолчанию
size - размер раздела в мегабайтах

Нужно помнить, что жо момента нажатия кнопки S никакие изменения не будут записаны на жёсткий диск.

Если были не записанные изменения, то выводится подсказка:
		[S]ave changes | [D]iscart changes
		
Для форматирования раздела необходимо выбрать размеченный раздел и нажать кнопку F:
		Selected volume [3]
		[f]ormat | [r]emove
		[C]reate
Форматирование не доступно, если есть несохраненные изменения в таблице разделов.

Также вы можете инсталлировать загрузчик совместимый с ERS(ZX-Evo) и xBIOS(ATM)
Для этого необходимо выбрать раздел(FAT) с установленной NedoOS и нажать кнопку B.
После этого будет записан непрерывный(не фрагментированный) файл zxldr на текущий выбранный раздел,
также будут записаны загрузочные сектора(2 и 3) на жесткий диск.
Если на жестком диске уже имеются(установленные is-dos'ом или tasis'ом) загрузочные сектора, то они не изменяются,
изменится лишь адрес загрузочного раздела и при запуске бутлоадера будет предоставлен выбор, что запускать.

Бутлоадер, при запуске, определяет наличие загрузочных разделов is-dos(TASIS) и выводит их список.
Так же выводится пункт запуска с жёсткого диска sd_boot.$c(OSATM2HD.$C в случае ATM).

Нужно помнить, что не допускается изменение, каким либо образом, файла zxldr, так как его физический адрес(в секторах)
используется при запуске с жёсткого диска. Но вы можете его переместить(переимоновать), куда либо в пределах текущего раздела, командой ren:
ren /ZXLDR /bin/не_трогать.bin
