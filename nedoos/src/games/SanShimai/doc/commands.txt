В тексте исполбзуются следующие команды: 

01          - #CLS                                       - Очистка тестового окна после нажатия
04 file     - #LOAD_SCENARIO "file"                      - Загрузка сценария file (название OVL файла без расширения)
06 X        - #LOAD_PICTURE "XXX"                        - Загрузка рисунка X
07 X        - #LOAD_MUSIC "XXX"                          - Выбор музыки X
08          - #WAIT                                      - Ожидание нажатия любой клавиши
09          - #WAITCRLF                                  - Ожидание нажатия любой клавиши и продолжение со следующей строки
0A          - ^                                          - Перенос на следующую строку

------------ группа работы с локальными переменными -------------
0B 01 X Y   - #LOCAL_LET [XX]=[XX]+YY                [XX] ячейка YY ЧИСЛО
0B 02 X Y   - #LOCAL_LET [XX]=[XX]+[YY]                XX,YY  -ЯЧЕЙКИ

0B 03 X Y   - #LOCAL_LET [XX]=[XX]-[YY]                                          -
0B 05 X Y   - #LOCAL_LET [XX]=YYY                  - Присвоение локальной переменной X=Y
0B 07 X Y Z - #LOCAL_LET [XX]=[YY]-ZZ              - ВЫЧИТАНИЕ
0B 08 X Y Z - #IF [YY]>0 AND [ZZ]>0 THEN [XX]=1    - Проверить ячейки Y и Z на истинность
0B 09 X Y Z - #IF [YY]>0 OR [ZZ]>0 THEN [XX]=1     - Проверить ячейки Y или Z на истинность
0B 0A X     - #IF [XX]<>0 THEN CONTINUE             - Продолжить если X<>0
0B 14 X Y   - #RANDOMIZE [XX]_YY                   - генерирует случайное число в ячейку
                                                       [XX] в диапазоне 0...YY-1
0B 32 X Y Z - #IF [XX]>0 THEN GOTO ZZYY            - Перейти по адресу YZ если X истинно
0B 33 X Y Z - #IF [XX]=0 THEN GOTO ZZYY            - Перейти по адресу YZ если X ложно
0B 34 X Y Z - #IF [XX]<0 THEN GOTO ZZYY            - Перейти по адресу YZ (ЕСЛИ Х<0 )

0B 35 X Y Z -  #IF [XX]>00 THEN GOTO ZZYY          - Перейти по адресу YZ если X истинно

0B 36 XY    - #GOTO XXYY                           - Перейти по адресу XY

------------ группа работы с глобальными переменными ------------
0C 01 X Y   - #GLOB_TO_LOC [XX]=[YY]                     - Перевод глобальной переменной в локальную LOC [XX]= GLOB [YY]
0C 02 X Y   - #LOC_TO_GLOB [XX]=[YY]                                         -
0C 03 X Y   - #GLOBAL_LET [XX]=YYY                       - Присвоение глобальной переменной X=Y
 
11 02 [XX] 06 [YY] - #LOAD_PICTURE [XX]+[YY]
for double screen output - грузит экраны [хx] и [yy],  экран [хx] ( в основном спрайт) рисуется
поверх экрана [yy] либо позволяет выводить на экран сразу два спрайта. способ прорисовки определяется рисунком [yy]                                      - 

------------ группа работы с меню ------------
12 01 X     - #MENU[XXX]=OFF                             - Отключить главное меню X
12 02 X     - #MENU[XXX]=ON                              - Включить главное меню X
12 04 X Y   - #SUBMENU[XXX,YYY]=OFF                      - Отключить в главном меню X субменю Y
12 05 X Y   - #SUBMENU[XXX,YYY]=ON                       - Включить в главном меню X субменю Y

0F          - #EXIT_LIGHT                                - Завершение работы вспышкой aka HAPPY END
10          - #EXIT_DARK                                 - Завершение работы затемнением aka GAME OVER
13 X        - #FLASH XXX                                 - Вспышка длительностью X
14 X        - #SLEEP XXX                                 - Пауза длительностью X
