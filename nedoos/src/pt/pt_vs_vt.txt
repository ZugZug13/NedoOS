Преимущества PT по сравнению с VT:

+ работает на реальном Спектруме (чистый звук, нет лагов, фреймово)
+ edit melody
+ "транспонирование" громкости
+ горячая кнопка помещения параметров ноты в autodata (SS+A)
+ горячие кнопки autodata (CS+A) и entoenv (CS+E)
+ кнопка "пустая нота" (F) = autodata+autoenv+scroll
+ прозрачный ввод огибающей (E). В этом режиме нота НЕПОРТИТСЯ. При AutoEnv=ON режим устойчивый, иначе выключаетсяпосле первой же ноты 
+ анализаторы, без которых трудно подогнать глисс
+ играет оба чипа по Enter
+ быстрый переход в начало-середину-конец текущего трека (Home, End)
+ установка autoscroll с клавиатуры (SS+цифра)
+ кнопка очистки ноты вместе с параметрами (SS+L)
+ кнопка очистки команды (K)
+ быстрое копирование треков целиком
+ merge модуля
+ весь сэмпл-орнамент можно редактировать горячими клавишами
+ сдвиг-раздвижение строк в сэмпле-орнаменте
+ переход к редактированию нужного сэмпла-орнамента с клавиатуры
+ ввод номера сэмпла-орнамента, куда копировать, тоже с клавиатуры
+ переход между паттернами/позициями с клавиатуры (PgUp, PgDn)
+ листание сэмплов-орнаментов с клавиатуры (PgUp, PgDn)
+ листание орнаментов в окне сэмпла и сэмплов в окне орнамента с клавиатуры (SS+U, SS+I)
+ копирование с наложением
+ одновременно доступно 3 октавы (SS+нота, нота, СS+нота)
+ бемоли (SS+Y)
+ TurboSound одним модулем
+ быстрое переключение между одинаковыми местами двух чипов (Extend=Tab)
+ интегрирован с ALASM для компиляции с плеером (в старых версиях есть и компилятор)
+ может сохранять без удаления "лишних" данных
+ можно быстро перевести трек (музыкальный канал) из Tempo=6 в Tempo=3:ss2, Home, Down, Ins (удерживать до получения результата)+ можно быстро перевести трек из Tempo=3 в Tempo=6:ss1, Home, Down, BackSpace (удерживать до получения результата) 
+ при запуске паттерны в ордере уже пронумерованы
+ при запуске уже созданы сэмплы E,F и орнаменты 1,2,3,4,5,6,F
+ команда tone offset
+ импорт некомпилированных модулей Sound Tracker, ST Pro
+ декомпилятор (сам ищет модуль в файле)
+ можно работать, не имея мыши (в VT нельзя работать без мыши и обязательно numpad)
+ в редакторе сэмплов тон вводится "накручиванием"
+ в редакторе сэмплов можно менять целые столбики проведением по ним курсора
+ окошко редактирования сэмпла больше, чем в VT
+ General Sound
+ версия с линейной таблицей громкости (для сверхкороткого плейера)
- в VT окошко паттерна выше
- в VT можно редактироть сэмплы-орнаменты во время проигрывания
- в VT можно загрузить и редактировать много модулей (но нельзя сделать merge)
- в VT раздельное включение фич каналов
- в VT есть Undo/Redo
- в VT есть копирование произвольного прямоугольного блока (но нет наложения и быстрого копирования треков целиком)
- в VT импорт модулей SQT, ASC, PSC, FLS, GTR, PT1, FXM, PSM (но нельзя импортировать
некомпилированные модули Sound Tracker, ST Pro)
- в VT можно паттерны длиннее 64 строк
- в VT можно больше 48 паттернов
- в VT можно протестировать ноту с огибающей в редакторе сэмплов
- в VT можно вводить все параметры цифрами без накручивания (но в редакторе сэмплов, где как раз нужно накручивание, его нет)
- в VT есть рендер куска мелодии в сэмпл
