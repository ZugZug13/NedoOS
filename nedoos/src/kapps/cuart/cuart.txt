Программа предназначена для  работы с ESP8266/ESP32 подключенной к COM-порту, но никто не запрещает использовать ее как плохой терминал
Размер буфера приема ~8192 байта, после достижения конца  буфер просто начнет перезаписываться сначала.

Кнопки:
    ESC     Выход
    BS      Удалить последний введенный символ
    CTRL+S  Сохранение буфера на диск в файл buffer.log
    F1      Выбор скорости порта 115200
    ......  ............
    F10     Выбор скорости порта 1200
    PgUp    Отправка "AT+GMR" для отображения версии ПО
    End     режим немедленной отправки символа после ввода    
    ALT+U   Отправка "AT+CIPUPDATE" для обновления версии ПО

Настройки порта хранятся в файле bin\browser\espcom.ini построчно

RBR_THR
IER
IIR_FCR     Настройки 16550 подобного порта, например ZXWiFi или Evolution
LCR	
MCR
LSR
MSR
SR
divider     Делитель определяет скорость порта, скорость зависит от кварца
comType     Тип порта 0 - Evolution, 1 - ATM2
