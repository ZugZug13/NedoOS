---

## Установка NedoOS на SD-карту (ZX Evolution)

1. Отформатируйте SD-карту в **FAT32**.
2. Скачайте [архив с релизом](https://github.com/ZugZug13/NedoOS/releases/download/nedoos-release-latest/nedoos-release.zip).
3. Распакуйте **все файлы в корень** SD-карты.
4. В бут-меню **ERS (baseconf)** выберите пункт **«SD card boot»**  
   или нажмите горячую клавишу **5**.
